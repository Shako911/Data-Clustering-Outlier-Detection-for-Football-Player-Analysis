# Data-Clustering-Outlier-Detection-for-Football-Player-Analysis
# Рџй рЃцрЃћрЃ«рЃЉрЃБрЃарЃЌрЃћрЃџрЃћрЃЉрЃўрЃА рЃљрЃюрЃљрЃџрЃўрЃќрЃў: DBSCAN рЃЎрЃџрЃљрЃАрЃбрЃћрЃарЃўрЃќрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃбрЃљрЃџрЃљрЃюрЃбрЃћрЃЉрЃўрЃА рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљ (EAI_HW2)

рЃћрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА **рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃљрЃюрЃљрЃџрЃўрЃќрЃўрЃА** рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџ рЃАрЃљрЃерЃўрЃюрЃљрЃЮ рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃўрЃќрЃюрЃљрЃЊ рЃўрЃАрЃљрЃ«рЃљрЃЋрЃА рЃцрЃћрЃ«рЃЉрЃБрЃарЃЌрЃћрЃџрЃћрЃЉрЃўрЃА рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃБрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЎрЃџрЃљрЃАрЃбрЃћрЃарЃўрЃќрЃљрЃфрЃўрЃљрЃА рЃЊрЃљ **рЃљрЃарЃљрЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў (рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў) рЃЏрЃЮрЃЌрЃљрЃЏрЃљрЃерЃћрЃћрЃЉрЃўрЃА** рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљрЃА.

рЃЕрЃЋрЃћрЃю рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃћрЃЌ **DBSCAN** рЃЎрЃџрЃљрЃАрЃбрЃћрЃарЃўрЃќрЃћрЃЉрЃўрЃА рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃў, рЃарЃљрЃЌрЃљ рЃЊрЃљрЃњрЃЋрЃћрЃЊрЃњрЃўрЃюрЃљ рЃЏрЃЮрЃЌрЃљрЃЏрЃљрЃерЃћрЃћрЃЉрЃўрЃА рЃбрЃўрЃърЃБрЃарЃў рЃ»рЃњрЃБрЃцрЃћрЃЉрЃў (рЃЎрЃџрЃљрЃАрЃбрЃћрЃарЃћрЃЉрЃў) рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃњрЃЋрЃћрЃЋрЃџрЃўрЃюрЃљ рЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃгрЃћрЃарЃбрЃўрЃџрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃљрЃарЃфрЃћрЃарЃЌ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊ рЃ»рЃњрЃБрЃцрЃерЃў рЃљрЃа рЃ«рЃЋрЃЊрЃћрЃЉрЃўрЃљрЃю (**Outlier-рЃћрЃЉрЃў**).

## ­Ъџђ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў рЃЊрЃљ рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў

* **рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃерЃћрЃЏрЃфрЃўрЃарЃћрЃЉрЃљ (Dimensionality Reduction):** **PCA (Principal Component Analysis)** рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА (PC1 рЃЊрЃљ PC2).
* **рЃЎрЃџрЃљрЃАрЃбрЃћрЃарЃўрЃќрЃљрЃфрЃўрЃљ:** **DBSCAN (Density-Based Spatial Clustering of Applications with Noise)** - рЃўрЃЊрЃћрЃљрЃџрЃБрЃарЃўрЃљ рЃљрЃарЃљрЃарЃћрЃњрЃБрЃџрЃљрЃарЃБрЃџрЃў рЃцрЃЮрЃарЃЏрЃўрЃА рЃЎрЃџрЃљрЃАрЃбрЃћрЃарЃћрЃЉрЃўрЃА рЃАрЃљрЃърЃЮрЃЋрЃюрЃћрЃџрЃљрЃЊ рЃЊрЃљ Outlier-рЃћрЃЉрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА.
* **рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃћрЃюрЃљ:** Python
* **рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃў:** Pandas, NumPy, Scikit-learn, Matplotlib/Seaborn.

## ­Ъј» рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃћрЃЉрЃў рЃЊрЃљ рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў

рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃЮрЃарЃўрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃАрЃљрЃЎрЃЋрЃљрЃюрЃФрЃЮ рЃћрЃбрЃљрЃърЃћрЃЉрЃќрЃћ, рЃарЃЮрЃњрЃЮрЃарЃф рЃћрЃА рЃљрЃдрЃгрЃћрЃарЃўрЃџрЃўрЃљ `EAI_HW2.ipynb` рЃюрЃЮрЃБрЃЌрЃЉрЃБрЃЦрЃерЃў:

1.  **DBSCAN-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ:** рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў $\epsilon$ рЃЊрЃљ $MinSamples$ рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃА рЃерЃћрЃарЃЕрЃћрЃЋрЃљ рЃЊрЃљ рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ.
2.  **рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ:** DBSCAN-рЃўрЃА рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃўрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ 2D PCA рЃАрЃўрЃЋрЃарЃфрЃћрЃерЃў, рЃАрЃљрЃЊрЃљрЃф рЃюрЃљрЃЌрЃџрЃљрЃЊ рЃЕрЃљрЃюрЃА рЃЎрЃџрЃљрЃАрЃбрЃћрЃарЃћрЃЉрЃў рЃЊрЃљ Outlier-рЃћрЃЉрЃў (Cluster = -1).
3.  **Outlier-рЃћрЃЉрЃўрЃА рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљ:** рЃЏрЃЮрЃЌрЃљрЃЏрЃљрЃерЃћрЃћрЃЉрЃўрЃА рЃАрЃўрЃўрЃА рЃерЃћрЃЊрЃњрЃћрЃюрЃљ, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃЏрЃЮрЃюрЃўрЃерЃюрЃБрЃџрЃўрЃљ рЃарЃЮрЃњрЃЮрЃарЃф Outlier-рЃћрЃЉрЃў.
4.  **рЃЉрЃўрЃќрЃюрЃћрЃА рЃљрЃюрЃљрЃџрЃўрЃќрЃў (рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃљрЃфрЃўрЃљ):** Outlier-рЃћрЃЉрЃўрЃА рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃўрЃА (рЃърЃЮрЃќрЃўрЃфрЃўрЃљ, рЃцрЃљрЃАрЃў, рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃљ) рЃљрЃюрЃљрЃџрЃўрЃќрЃў.

### ­ЪњА рЃЉрЃўрЃќрЃюрЃћрЃА рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃљ

рЃцрЃћрЃ«рЃЉрЃБрЃарЃЌрЃўрЃА рЃЎрЃџрЃБрЃЉрЃўрЃА рЃърЃћрЃарЃАрЃърЃћрЃЦрЃбрЃўрЃЋрЃўрЃЊрЃљрЃю, рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃдрЃўрЃарЃћрЃЉрЃБрЃџрЃў Outlier-рЃћрЃЉрЃў рЃљрЃарЃўрЃљрЃю:

> **"рЃўрЃљрЃцрЃў рЃЊрЃљ рЃФрЃџрЃўрЃћрЃарЃў"** рЃЏрЃЮрЃЌрЃљрЃЏрЃљрЃерЃћрЃћрЃЉрЃў, рЃљрЃю **рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃЏрЃЦрЃЮрЃюрЃћ** рЃЏрЃЮрЃЌрЃљрЃЏрЃљрЃерЃћрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџ рЃЎрЃџрЃљрЃАрЃбрЃћрЃарЃћрЃЉрЃерЃў рЃљрЃа рЃ«рЃЋрЃЊрЃћрЃЉрЃўрЃљрЃю. рЃћрЃА рЃЏрЃЮрЃЌрЃљрЃЏрЃљрЃерЃћрЃћрЃЉрЃў рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃћрЃю **рЃЊрЃљрЃБрЃцрЃљрЃАрЃћрЃЉрЃћрЃџ рЃбрЃљрЃџрЃљрЃюрЃбрЃћрЃЉрЃА** рЃЊрЃљ рЃЉрЃљрЃќрЃљрЃарЃќрЃћ рЃЏрЃЮрЃЏрЃњрЃћрЃЉрЃўрЃљрЃюрЃў рЃерЃћрЃюрЃљрЃФрЃћрЃюрЃўрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃљрЃА.

## ­ЪЊЮ рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЋрЃБрЃерЃЋрЃљрЃЌ рЃърЃарЃЮрЃћрЃЦрЃбрЃў

1.  **рЃЎрЃџрЃЮрЃюрЃўрЃарЃћрЃЉрЃљ:** рЃЊрЃљрЃљрЃЎрЃџрЃЮрЃюрЃўрЃарЃћрЃЌ рЃћрЃА рЃарЃћрЃърЃЮрЃќрЃўрЃбрЃЮрЃарЃў GitHub-рЃЊрЃљрЃю.
    ```bash
    git clone [рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃарЃћрЃърЃЮрЃќрЃўрЃбрЃЮрЃарЃўрЃА рЃџрЃўрЃюрЃЎрЃў]
    ```
2.  **рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ:** рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃњрЃљрЃЦрЃЋрЃЌ рЃАрЃљрЃГрЃўрЃарЃЮ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃў.
    ```bash
    pip install pandas numpy scikit-learn matplotlib seaborn
    ```
3.  **рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ:** рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ `EAI_HW2.ipynb` рЃцрЃљрЃўрЃџрЃў Jupyter Notebook-рЃерЃў рЃљрЃю VS Code-рЃерЃў рЃЊрЃљ рЃЏрЃўрЃ░рЃДрЃћрЃЋрЃўрЃЌ рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃА.
